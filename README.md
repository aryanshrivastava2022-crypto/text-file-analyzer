# Text File Analyzer
A Python application that analyzes text files and provides useful statistics.

# Features
- Count characters
- Count words
- Count lines
- Find most frequent words
- Simple command-line interface

# Requirements
- Python 3.x

# How to Run
1. Create or place a text file in the project folder.
2. Run:
text_file_analyzer.py
3. Enter the file name when prompted.

# Sample Output
📊 Text File Analysis
Characters: 250
Words: 45
Lines: 8
🔝 Top 5 Most Frequent Words:
python: 5
code: 4
project: 3
